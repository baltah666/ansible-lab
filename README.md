# Ansible Lab

This repository contains Ansible automation scripts for managing network devices in my home lab.

## Contents:

- **install_tools.yml**: Ansible playbook for installing network troubleshooting tools.
- **inventory.ini**: Inventory file for the devices in the network.
- **ansible.cfg**: Configuration file for Ansible.
